# ChainQA部署说明

## 预先准备

- 部署区块链和IPFS，请参考 `区块链和IPFS部署.md`
- 部署好区块链后，完成合约部署，请参考 `合约部署.md`

## 如何启动

请先完成预先准备环节

### 第一步：前端环境打包

要求环境：node>20

- 打包：

```bash
cd ./front
yarn build
```

> 打包生成的结果位于/front/dist下

- 将打包结果放到/back/dist下。

### 第二步：后端启动运行

要求缓解：go>=1.23.0，推荐1.24.0

运行：

```bash
go run main.go
```

后端监听9000端口，后访问 `localhost:9000`即可
